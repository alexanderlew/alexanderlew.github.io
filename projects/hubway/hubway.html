Hubway - Bike Sharing in Boston | open stations

This visualization is a reworking of a homework project for CS 171 Visualization. This is a real-time system map for Boston's Hubway bike share system. The slices of each circle represent the number of available bikes and empty docks at a station, while the entire circle represents the total station capacity. Tools used in this visualization include D3 (proportional pie charts and tool tips), Leaflet (maps), and the Hubway real-time data (data).
