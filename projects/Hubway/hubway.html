Hubway - Bike Sharing in Boston | open stations

This visualization is based on a homework project for CS 171 Visualization. This is a system status map for Boston's Hubway bike share system. The circle size indicates the full capacity of the station, while the slices of the circle show the number of docks and bikes available. Tools used in this visualization were D3 (pie charts and tool tips), Leaflet (maps), and the Hubway real time data (data).
